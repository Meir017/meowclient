getComments

Gets the comments on the project.

Parameters

  • offset (number) The offset of comments to return
  • limit (number) The limit of comments to return

Returns

This fetches and returns the data from the project comments - for example from https://api.scratch.mit.edu/users/griffpatch/projects/612229554/comments?offset=0&limit=20. Type definitions will be added soon.
