# ScratchSession
The main part of meowclient.

## Functions
### getProject

Gets a project

#### Parameters
- `id` (`number`) - The ID of the project you want to get.

#### Returns
This returns a `Project` class.

### getProfile

Gets a profile

#### Parameters
- `username` (`string`) - The username of the profile you want to get.

#### Returns
This returns a `Profile` class.
